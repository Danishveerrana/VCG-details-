Voice-Controlled Graphics (VCG)

VCG is a cutting-edge Python-based system that lets users draw 2D graphics using voice commands. It utilizes AI-driven speech recognition and turtle graphics to create a hands-free visual programming environment.

Overview

  • Draw shapes like squares, circles, lines with voice
  • Powered by Python, Google Speech API & Turtle module
  • Great tool for education, creativity & accessibility

Project Goals

  • Empower users to draw using only speech
  • Assist physically challenged individuals
  • Encourage integration of AI with art
  • Promote interactive, accessible programming

My Contributions

  • Improved voice command handling
  • Fixed number conversion bugs (e.g., “ten” → 10)
  • Implemented new commands: undo, clear, save
  • Added logging and real-time debugging support
  • Created a simple help assistant inside the terminal
  • Enhanced error handling so users are informed when recognition or execution fails, improving comfort & usability

Key Features

  • Natural-language voice commands
  • Undo, save & clear functions for flexibility
  • Responsive to continuous dictation
  • Supports learning shapes and basic graphics coding

Impact

  • Can be used in e-learning platforms

Future Scope
